# Frontend

Application React (Vite) permettant de gerer les sessions de jeux de societe.

## Commandes

- `pnpm install` : installer les dependances du workspace depuis la racine.
- `pnpm dev` : lancer le serveur de developpement (port 5173 par defaut).
- `pnpm build` : construire l'application.
- `pnpm preview` : servir le build localement.

Les variables d'environnement attendues sont referencees dans `.env.example`.
